Технологический стек для предоставленного кода использует такие компоненты:

Язык программирования: Python
Фреймворк для работы с Telegram API: Pyrogram
Модули и библиотеки: pyrogram, urllib, BeautifulSoup
Интеграция с Telegram API: экземпляр класса Client из Pyrogram используется для взаимодействия с API Telegram
Внешний файла config.txt используется для хранения API ID и API Hash
Веб-скрапинг: urllib для получения содержимого веб-страницы, BeautifulSoup для парсинга HTML
Работа с сообщениями: фильтры filters из Pyrogram для обработки команд и сообщений от пользователей
Обработчики сообщений: использование декораторов client.on_message для определения функций-обработчиков для команды /start и поиска
Глобальная переменная is_searching для отслеживания состояния поиска
Отправка сообщений: использование метода send_message из объекта Client для отправки сообщений пользователю
